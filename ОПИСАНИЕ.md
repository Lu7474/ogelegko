# Платформа для подготовки к ОГЭ и ЕГЭ по математике

> Веб-приложение на Django для проведения пробных экзаменов в школе.
> Поддерживает три формата: **ОГЭ**, **ЕГЭ профиль**, **ЕГЭ база**.

## 🎓 Для учеников

### Вход в систему

- Авторизация по **полному имени** и **паролю**
- Защита от подбора пароля: блокировка IP после нескольких неверных попыток
- Безопасное хранение паролей (Django password hashing)

---

### Выбор варианта

После входа ученик видит список **доступных вариантов**, отфильтрованных по типу экзамена его класса.
Для каждого варианта отображается:

| Поле | Описание |
|------|----------|
| Номер варианта | Например: `092` |
| Количество заданий | Зависит от типа экзамена |
| Использовано попыток | Из максимально допустимого числа |
| Статус | Доступен / Лимит исчерпан |

> Если лимит попыток исчерпан — вариант недоступен для повторного прохождения.

---

### Прохождение экзамена

- Все задания варианта отображаются на **одной странице** — можно листать свободно
- Каждое задание может содержать:
  - Текст с математическими формулами и таблицами
  - Изображение (фото условия, чертёж и т.д.)
  - **Подсказку формата ответа** (целое число / дробь / последовательность через `;`)
- **Автосохранение** ответов при каждом вводе — данные не потеряются при случайном закрытии вкладки
- Таймер обратного отсчёта отображает оставшееся время:
  - ОГЭ — 3 ч 55 мин
  - ЕГЭ профиль — 3 ч 55 мин
  - ЕГЭ база — 3 ч

---

### Завершение и результаты

После нажатия **«Завершить»** открывается окно подтверждения.
Сразу после сдачи ученик видит развёрнутые результаты:

```
Первичный балл:   18 / 25
Оценка:           4
Правильных:       18
Неправильных:     7
Время:            1ч 42мин
```

Дополнительно отображаются **цветные точки** по каждому заданию:
- 🟢 Зелёная — правильный ответ
- 🔴 Красная — неправильный ответ

Для каждого неправильного ответа показано:
- Что ответил ученик
- Правильный ответ

---

### Личный кабинет

Ученик может в любой момент открыть профиль и увидеть:

- Все завершённые попытки с датой, баллом и оценкой
- Детальный разбор любой попытки (какие задания верны/неверны)
- Историю своих экзаменов

## 🛠 Для администратора

Отдельный раздел с защищённым входом `/admin/` для учителей и организаторов.

---

### Панель управления (Dashboard)

Главная страница даёт **сводку по всей школе**:

- Общее количество классов, учеников, вариантов, попыток
- Таблица по каждому активному классу:
  - Количество учеников
  - Число попыток
  - Средний результат (%)

---

### Управление классами

| Действие | Описание |
|----------|----------|
| Создать класс | Название + тип экзамена (ОГЭ / ЕГЭ профиль / ЕГЭ база) |
| Редактировать | Изменить название или тип экзамена |
| Архивировать | Скрыть класс без удаления (переключатель активности) |
| Удалить | Полное удаление со всеми учениками |
| Статистика класса | Таблица по каждому ученику: средний %, число попыток, последняя попытка |

---

### Управление учениками

| Действие | Описание |
|----------|----------|
| Добавить вручную | ФИО + пароль + класс |
| Массовый импорт | Загрузить `.xlsx` файл со списком учеников |
| Редактировать | Изменить ФИО, пароль или класс |
| Удалить | Удалить ученика со всеми его попытками |
| Статистика ученика | График динамики результатов + таблица всех попыток |

#### Импорт учеников из Excel

Формат файла:

```
ФИО                      | Пароль | Класс
Иванов Иван Иванович     | 1234   | 9А
Петрова Мария Сергеевна  | 5678   | 9А
```

---

### Управление вариантами

| Действие | Описание |
|----------|----------|
| Создать вариант | Номер + тип экзамена + лимит попыток + задания |
| Редактировать | Изменить любое поле, добавить/удалить задания |
| Активировать / Скрыть | Моментальное переключение видимости для учеников |
| Дублировать | Создать копию варианта (сохраняет лимит попыток и темы) |
| Удалить | Полное удаление варианта |
| Статистика варианта | Процент правильных ответов по каждому заданию |
| Импорт из JSON | Загрузить вариант из файла |

#### Настройки варианта

- **Номер варианта** — произвольная строка (например `092`, `ЕГЭ-2024-1`)
- **Тип экзамена** — определяет таблицу перевода баллов
- **Лимит попыток** — максимальное число раз, которое ученик может пройти вариант (0 = без ограничений)

#### Конструктор заданий

Каждое задание содержит:

| Поле | Описание |
|------|----------|
| Текст | Условие задачи (поддерживает HTML, таблицы, формулы) |
| Изображение | Загрузка фото или чертежа |
| Правильный ответ | Строка (число, дробь, последовательность) |
| Баллы | Вес задания в первичном балле |
| Источник | ФИПИ / Распечатай и реши / Динамика / Вручную |
| Тема | Алгебра / Геометрия / Вероятность / Функции / Уравнения / Теория чисел / Практика |

---

### Статистика ученика

Детальная страница с:

- Средним результатом по всем попыткам (%)
- **Линейным графиком** динамики успеваемости (Chart.js)
- Таблицей попыток с датой, баллом, оценкой и временем
- Возможностью **удалить отдельную попытку**

Выгрузка всех результатов в **Excel (.xlsx)**:

| Колонка | Содержимое |

| ФИО | Полное имя ученика |

## 🧮 Умная проверка ответов

Система автоматически нормализует ответ ученика перед сравнением:

```
"3,5"   →  3.5     ✅ совпадает с "3.5"
"1/2"   →  0.5     ✅ совпадает с "0.5"
"0.25"  →  0.25    ✅ совпадает с "1/4"
"  7  " →  7       ✅ совпадает с "7"
```

## 🔒 Безопасность

- **IP-блокировка** после нескольких неверных попыток входа (для учеников и администраторов раздельно)
- **CSRF-защита** на всех формах
- **Раздельная аутентификация**: ученики и администраторы — разные системы входа
- Пароли хранятся в виде **хэшей** (PBKDF2)
- Все страницы ученика защищены редиректом на логин
- Все страницы администратора проверяют `is_staff`
- Выход только через POST-запрос (защита от CSRF-атак через ссылки)
