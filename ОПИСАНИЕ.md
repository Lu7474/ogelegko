# Платформа для подготовки к ОГЭ и ЕГЭ по математике

Веб-приложение для проведения пробных экзаменов в школе.
Учитель создаёт варианты — ученики решают онлайн, результаты сохраняются автоматически.

Поддерживаются три формата: **ОГЭ**, **ЕГЭ профиль**, **ЕГЭ база**.

---

## Для учеников

### Вход

Ученик входит по **полному имени** и **паролю**, которые выдаёт учитель.
После нескольких неверных попыток вход временно блокируется.

---

### Выбор варианта

После входа ученик видит список доступных вариантов.
Для каждого варианта отображается:
- Номер варианта
- Количество заданий
- Сколько попыток уже использовано
- Доступен ли вариант для прохождения

---

### Прохождение экзамена

- Все задания отображаются на одной странице — можно листать в любом порядке
- Каждое задание может содержать текст, одно или несколько изображений и подсказку формата ответа
- Задания могут иметь общее условие (текст/картинка перед группой заданий)
- **Ответы сохраняются автоматически** — если закрыть вкладку, прогресс не потеряется
- Таймер показывает оставшееся время:
  - ОГЭ — 3 часа 55 минут
  - ЕГЭ профиль — 3 часа 55 минут
  - ЕГЭ база — 3 часа

---

### Результаты

После завершения ученик сразу видит:
- Первичный балл и оценку
- Количество правильных и неправильных ответов
- Затраченное время
- По каждому заданию — что ответил и какой был правильный ответ

---

### Повторить ошибки

После просмотра результатов можно нажать **«Повторить ошибки»** — система создаёт
отдельный вариант только из тех заданий, на которые был дан неправильный ответ.
Это удобно для отработки слабых мест.

---

### Личный кабинет

Ученик может открыть свой профиль и посмотреть:
- Все прошлые попытки с датой, баллом и оценкой
- Динамику результатов на графике
- По каким номерам заданий результат ниже 50%
- Подробный разбор каждой попытки

---

## Для учителя

Отдельная панель управления с входом по логину и паролю.

---

### Главная страница (Дашборд)

Сводка по всей школе:
- Количество классов, учеников, вариантов и попыток
- По каждому классу — число учеников, попыток и средний результат
- Последние 10 завершённых попыток
- Уведомление, если есть попытки с заданиями, ожидающими ручной проверки

---

### Классы

- Создать класс с указанием типа экзамена (ОГЭ / ЕГЭ профиль / ЕГЭ база)
- Скрыть класс, не удаляя его
- Посмотреть статистику по каждому ученику класса: средний результат, число попыток, дата последней
- Аналитика по номерам заданий: для каждого задания — сколько раз решали и процент верных ответов
- Экспорт аналитики класса в Excel
- Выбрать несколько попыток для массовой проверки ручных заданий

---

### Ученики

- Добавить ученика вручную (имя, пароль, класс)
- Загрузить список учеников из файла Excel
- Редактировать данные или удалить ученика
- Посмотреть историю попыток и динамику результатов конкретного ученика
- Аналитика по номерам заданий: выявить слабые темы

---

### Варианты

- Создать вариант вручную — добавить задания, правильные ответы, изображения (основное + несколько дополнительных)
- Импортировать задания из каталога
- Включить или скрыть вариант для учеников в один клик
- Скопировать существующий вариант
- Скачать вариант в формате Word — версия для учителя (с ответами) и для ученика (без ответов)
- Скачать несколько вариантов одним ZIP-архивом (Word-файлы для печати)
- **Архив ZIP** — экспортировать варианты в переносимый архив с заданиями и изображениями; импортировать на другую установку; при конфликте номеров старый вариант не перезаписывается
- Задать лимит попыток (или без ограничений)
- Посмотреть статистику: по каждому заданию — процент правильных ответов

---

### Ручная проверка

Задания с развёрнутым ответом (часть 2 ЕГЭ профиль) учитель проверяет вручную прямо в панели.
Система уведомляет о новых завершённых попытках, которые ожидают проверки.
Доступна массовая проверка сразу нескольких попыток.

---

### Экспорт результатов

Выгрузка всех результатов в таблицу Excel:
- ФИО ученика, класс, тип экзамена
- Номер варианта, дата сдачи
- Балл, максимальный балл, оценка, время

Можно отфильтровать по классу или по варианту.

---

### Каталог заданий

Единая база заданий, из которой собираются варианты.
- Добавить задание вручную или импортировать
- Импорт с **СдамГИА**, из банка **ФИПИ** и из **PDF-файлов**
- Общее условие к заданию (текст и/или картинка)
- Несколько дополнительных изображений на одно задание
- Фильтрация по типу экзамена, номеру задания, источнику, ключевому слову
- Раздел «Неопределённые задания»: задания без номера и без ответа (с фильтром по источнику)
- История импортов с возможностью удаления записи или записи вместе с заданиями

---

## Проверка ответов

Система сама разбирается с форматом — не нужно вводить ответ строго определённым образом:

- `3,5` и `3.5` считаются одинаково верными
- `1/2` и `0.5` — тоже
- Лишние пробелы игнорируются
- Несколько допустимых вариантов ответа разделяются через `|` (любой засчитается)

### Оценки (2025/2026)

**ОГЭ математика:**

| Первичный балл | Оценка |
|:--------------:|:------:|
| 0 – 7          | 2      |
| 8 – 14         | 3      |
| 15 – 21        | 4      |
| 22 – 32        | 5      |

**ЕГЭ база:**

| Первичный балл | Оценка |
|:--------------:|:------:|
| 0 – 6          | 2      |
| 7 – 11         | 3      |
| 12 – 16        | 4      |
| 17 – 21        | 5      |

**ЕГЭ профиль:**
Первичный балл (0–30) переводится в тестовый (0–100) по официальной шкале ФИПИ.
